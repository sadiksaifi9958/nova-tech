# NovaTech 🚀

A modern, multi-page business website built with vanilla HTML, CSS, and JavaScript.

## 🌐 Live Demo
[View Live Site](https://sadiksaifi9958.github.io/NovaTech/)

## ✨ Features
- Responsive design — works on mobile, tablet and desktop
- Mobile hamburger menu
- Modal contact form with JavaScript injection
- Scroll animations using IntersectionObserver API
- Active nav link highlighting based on current page
- Dark / light mode toggle with localStorage persistence
- Contact form validation with error messages
- Price card selection interaction
- Smooth CSS transitions throughout

## 🛠️ Tech Stack
- HTML5
- CSS3 — Flexbox, CSS Variables, Media Queries
- JavaScript — DOM Manipulation, Web APIs, localStorage
- Font Awesome — icons

## 💡 What I Learned
Built this project to master JavaScript DOM manipulation and real website functionality. Learned how to use IntersectionObserver, localStorage, dynamic HTML injection with insertAdjacentHTML, CSS specificity, and how to debug JavaScript using browser DevTools.
